GNU MCU Eclipse News

Eclipse Embedded CDT plug-ins v5.1.1-202007271621 released

Jul 28, 2020 - Version 5.1.1 is the first release after the project migration to the Eclipse Foundation.

GNU MCU Eclipse plug-ins v4.7.2-202001271244 released

Jan 27, 2020 - Version 4.7.2-202001271244 is a maintenance release to fix the issue related to the change introduced by the update to https for the CMSIS Pack index.

GNU MCU Eclipse plug-ins v4.7.1-201911052135 released

Nov 6, 2019 - Version 4.7.1-201911052135 is a maintenance release; the main change is an enhancement in the logic to update the list of CMSIS Packs.

The GNU MCU Eclipse project has new forums

Sep 25, 2019 - The GNU MCU Eclipse forum was relocated from the Element14 site to a dedicated section in the xPack forum.

GNU MCU Eclipse plug-ins v4.6.1-201909231407 released

Sep 23, 2019 - Version 4.6.1-201909231407 is a new release; the main changes are the addition of support for the new xPack Dev Tools and performance fixes for the CMSIS Packs manager.

GNU MCU Eclipse ARM Embedded GCC v8.2.1-1.7 20190524 released

May 24, 2019 - Version v8.2.1-1.7 20190524 is a maintenance release of GNU MCU Eclipse ARM Embedded GCC that that removes the illegal links present in the previous releases.

GNU MCU Eclipse RISC-V Embedded GCC v8.2.0-2.2 20190521 released

May 21, 2019 - Version v8.2.0-2.2-20190521 is a maintenance release of GNU MCU Eclipse RISC-V Embedded GCC, that fixes the previous bug affecting GDB on macOS.

GNU MCU Eclipse QEMU v2.8.0-6 20190517 released

May 17, 2019 - Version 2.8.0-6 20190517 is a maintenance release, to fix the bug affecting the graphical mode in GNU/Linux, introduced in the previous release.

GNU MCU Eclipse ARM Embedded GCC v8.2.1-1.6 20190510 released

May 10, 2019 - Version v8.2.1-1.6 20190510 is a maintenance release of GNU MCU Eclipse ARM Embedded GCC that (finally) fixes the bugs affecting Windows LTO builds, present in the previous release.

GNU MCU Eclipse ARM Embedded GCC v8.2.1-1.5 20190426 released

Apr 26, 2019 - Version v8.2.1-1.5 20190426 is a maintenance release of GNU MCU Eclipse ARM Embedded GCC that fixes the bugs affecting Windows LTO builds, present in the previous release.
