GNU MCU Eclipse News

GNU MCU Eclipse RISC-V Embedded GCC v7.2.0-3-20180506 released

May 6, 2018 - Version v7.2.0-3-20180506 is a maintenance release of GNU MCU Eclipse RISC-V Embedded GCC; it updates to the latest RISC-V sources and the build process was improved, to add gdb-py and to support even older GNU/Linux machines and processors.

GNU MCU Eclipse Windows Build Tools v2.11-20180428 released

Apr 28, 2018 - Version 2.11-20180428 is a maintenance release of the GNU MCU Eclipse Windows Build Tools.

GNU MCU Eclipse plug-ins v4.3.3-201804191501 released

Apr 19, 2018 - Version 4.3.3-201804191501 is a maintenance release; the main changes are bug fixes and enhancements.

GNU MCU Eclipse ARM Embedded GCC v7.2.1-1.1 20180401 released

Apr 1, 2018 - Version v7.2.1-1.1 20180401 is a new release of GNU MCU Eclipse ARM Embedded GCC.

GNU MCU Eclipse ARM Embedded GCC v6.3.1-1.1 20180331 released

Mar 31, 2018 - Version v6.3.1-1.1 20180331 is the first release of GNU MCU Eclipse ARM Embedded GCC.

GNU MCU Eclipse plug-ins v4.3.2-201801250917 released

Jan 25, 2018 - Version 4.3.2-201801250917 is a maintenance release; the main addition is support for semihosting in the SiFive RISC-V project template.

GNU MCU Eclipse OpenOCD v0.10.0-7-20180123 released

Jan 23, 2018 - Version v0.10.0-7-20180123 is a maintenance release; the main change was to add semihosting support for RISC-V targets.

GNU MCU Eclipse RISC-V Embedded GCC v7.2.0-2-20180111 released

Jan 12, 2018 - Version v7.2.0-2-20180111 is a maintenance release of GNU MCU Eclipse RISC-V Embedded GCC; it updates to the latest RISC-V sources and the build process was improved, to support even older GNU/Linux machines and processors.

GNU MCU Eclipse OpenOCD v0.10.0-6-20180112 released

Jan 12, 2018 - Version v0.10.0-6-20180112 is a maintenance release; the main change was to update to the latest commits.

GNU MCU Eclipse plug-ins v4.3.1-201801092051 released

Jan 10, 2018 - Version 4.3.1-201801092051 is a new release; the main additions are: improved debugging via a detailed peripheral viewer, now also available for debugging RISC-V applications, and better integration with the xPack binary tools (RISC-V toolchain, OpenOCD).
