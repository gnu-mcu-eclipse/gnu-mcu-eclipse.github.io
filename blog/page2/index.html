GNU MCU Eclipse Windows Build Tools v2.12 20190422 released

GNU MCU Eclipse ARM Embedded GCC v8.2.1-1.4 20190214 released

Feb 14, 2019 - Version v8.2.1-1.4 20190214 is a maintenance release of GNU MCU Eclipse ARM Embedded GCC that fixes the bugs affecting Windows LTO builds, present in the previous release.

GNU MCU Eclipse QEMU v2.8.0-4-20190211 released

Feb 11, 2019 - Version 2.8.0-4-20190211 is a maintenance release, to fix a bug in the STM32F GPIO emulation.

GNU MCU Eclipse ARM Embedded GCC v8.2.1-1.3 20190202 released

Feb 2, 2019 - Version v8.2.1-1.3 20190202 is a maintenance release of GNU MCU Eclipse ARM Embedded GCC that fixes the GDB and liblto_plugin bugs present in the previous release.

GNU MCU Eclipse ARM Embedded GCC v8.2.1-1.2 20190119 released

Jan 19, 2019 - Version v8.2.1-1.2 20190119 is a maintenance release of GNU MCU Eclipse ARM Embedded GCC that fixes the 32-bit objcopy bug present in the previous release.

GNU MCU Eclipse OpenOCD v0.10.0-11-20190118 released

Jan 18, 2019 - Version v0.10.0-11-20190118 is a maintenance release; it updates to the latest master and removes the RISC-V patches.

GNU MCU Eclipse ARM Embedded GCC v8.2.1-1.1 20190102 released
