GNU MCU Eclipse News

GNU MCU Eclipse Windows Build Tools v2.9-20170629 released

Jun 29, 2017 - Version 2.9-20170629 is a maintenance release of the GNU MCU Eclipse Windows Build Tools.

GNU ARM Eclipse plug-ins update sites migrated to Bintray

May 2, 2017 - After serving almost 6 million downloaded files, the Eclipse update sites were migrated from the venerable SourceForge to the new Bintray site.

GNU ARM Eclipse plug-ins v3.4.1-201704251808 released

Apr 25, 2017 - Version 3.4.1-201704251808 is a maintenance release, intended to update the plugins to the new CMSIS Packs specs and to also fix some bugs.

GNU ARM Eclipse OpenOCD v0.10.0-20170418* released

Apr 19, 2017 - Version v0.10.0-201704182147 is a maintenance release. It updates libhid.a to the latest available version, 0.8.0-rc1. It includes binaries for Windows, macOS and GNU/Linux.

GNU ARM Eclipse plug-ins v3.3.1-201702251311 released

Feb 25, 2017 - Version 3.3.1-201702251311 is a new release; the main change is adding support for GCC 6.x, including new Cortex-M23/M33.

GNU ARM Eclipse plug-ins: Received fatal alert: handshake_failure

Jan 29, 2017 - For historical reasons, the GNU ARM Eclipse update sites for installing new software are hosted by SourceForge, and, up to now, they successfuly handled more than 5.000.000 downloaded files for this project, but in the last months SourceForge experienced several security issues affecting GNU ARM Eclipse plug-ins users.

GNU ARM Eclipse: 5.000.000 SourceForge downloads, January 2017

Jan 26, 2017 - January 2017 came with a nice surprise, the SourceForge statistics for the GNU ARM Eclipse project show more than 5.000.000 downloads in the last 4 years:

GNU ARM Eclipse OpenOCD v0.10.0-20170124* released

Jan 24, 2017 - Version v0.10.0-201701241841 is a major stable release. It includes binaries for Windows, macOS and GNU/Linux.

GNU ARM Eclipse plug-ins v3.2.1-201701141320 released

Jan 14, 2017 - Version 3.2.1-201701141320 is a new release; the main change is adding support for Eclipse Neon.2 (CDT 9.2).

GNU ARM Eclipse QEMU v2.8.0-20161227* released

Dec 27, 2016 - Version 2.8.0-201612271623 is a new release, adding support for active graphical buttons to all supported boards.
